# üöÄ Large-Scale Data Pipeline Migration from Mainframe to Hadoop

This project demonstrates how to migrate a **legacy mainframe data warehouse** to a **modern Hadoop ecosystem** using open-source Big Data tools such as **Hadoop, Spark, Hive, Sqoop, Oozie, and MySQL**.

---

## üß≠ Project Overview

This project simulates the migration of enterprise mainframe data to Hadoop for scalable storage and analytics.  
It automates the **data ingestion, transformation, and validation** processes using a modern Big Data stack.

### Objectives:
- Extract and migrate data from MySQL (mainframe source) to Hadoop (HDFS)
- Transform and enrich data using PySpark
- Store and query data efficiently using Hive
- Automate pipeline using Oozie
- Validate data integrity post-migration

---

## üß∞ Technologies Used

| Component | Technology |
|------------|-------------|
| Data Storage | Hadoop (HDFS) |
| Processing | Apache Spark (PySpark) |
| Query Engine | Apache Hive |
| Data Ingestion | Apache Sqoop |
| Workflow Scheduler | Apache Oozie |
| Source Database | MySQL |
| Validation | Python scripts |
